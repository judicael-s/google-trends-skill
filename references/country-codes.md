**Notes:**
- `--geo` accepts ISO 3166-1 alpha-2 codes (uppercase)
- `--hl` accepts BCP 47 language tags — the format is `language-REGION`
- For Belgium and Switzerland, choose the `hl` code matching the language community you're targeting (e.g., `fr-BE` for French-speaking Belgium, `nl-BE` for Dutch-speaking Belgium)
- Omitting `--geo` returns worldwide data
